>### ***pixCom***
>_(Pixel Communication)_
>
>Read and write binary (or text) data to pixels in an image.

**Usage:**

First "Load Image" file. (Needed for image dimensions.)  
Than "Read Bytes".  
If there is a message, you will see it.  
Edit Bytes or message.  
"Write Bytes".  
"Save Image".
